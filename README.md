# ELEC5305 Project: Adaptive Hybrid Speech Denoising

## Author: Tengfei Wang (540542743)

### Description
This project implements classical Wiener and Spectral Subtraction methods combined into an Adaptive Hybrid filter for speech denoising.

### Run Instructions
1. Open MATLAB
2. Set path to project folder
3. Run `main_hybrid_denoise.m`

Results and demo video will be saved in `/results/`.
